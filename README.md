# Ankit Parekh - AI Frontend Portfolio

Personal portfolio and resume site for [Ankit Parekh](https://github.com/AnkitParekh007), focused on AI frontend architecture, enterprise Angular, TypeScript, intelligent UI systems, copilots, RAG workflows, MCP/tool execution, and UI-aware agent experiences.

**Live site:** [ankitparekh007.github.io/resume](https://ankitparekh007.github.io/resume/)

## AI Frontend Focus

I build frontend systems where AI is part of the product experience, not just a chat widget.

- Embedded copilots for enterprise Angular applications
- Streaming AI response interfaces and structured message UX
- RAG citation cards, source previews, and context-aware answers
- MCP/tool-calling timelines and action execution states
- Human approval flows for sensitive or irreversible actions
- UI-aware agents that inspect page context and guide workflows
- Enterprise frontend architecture with Angular, TypeScript, RxJS, and reusable component systems
